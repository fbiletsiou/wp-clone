## Getting Started

### Dependencies

* Windows 10
* [Visual Studio Code ](https://code.visualstudio.com/)
* [Android Studio](https://developer.android.com/studio)
* [Node.js](https://nodejs.org/)

### Installing

* `cd C:\Users\<USER NAME>\Documents\wp-clone\`
* `npm install`
* `npm install --location=global expo-cli`
